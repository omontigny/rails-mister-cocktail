# Le Wagon MisterCocktail

## Context
This application is done while [Le wagon](https://lewagon.com) Bootcamp. It's a little application to manage cocktail receipt.

## Install
```
bundle install
yarn install
rails db:create
rails db:migrate
rails db:seed
```
## Ruby on Rails Version
` rails 6.1.4.4`
